A. Кружок \[0-0]/. Best teacher MAX

ограничение по времени на тест: 1 секунда
ограничение по памяти на тест: 256 Мбайтов
Условие

В кружке олимпиадного программирования, который проходит в лучшей школе мира есть самый лучший учитель Макс (он же Максим, он же Крутой). На первом занятии кружка ученики с Максом договорились время от времени проводить соревнования кто больше решит олимпиадных задач за полтора часа. Сегодня все пришли на занятие кружка, на котором будет такое соревнование. В конце соревнования Максу стали известны баллы каждого участника. Максу необходимо определить 1-го победителя (и подарить ему ноутбук), а так же сколько то призёров (чтобы подарить им по 100 рублей). Призеры - это все участники, которые набрали больше баллов, чем среднее арифметическое баллов всех участников.

Входные данные

В первой строке вводится число n - количество участников соревнования (3 ≤ n ≤ 1000) В следующих n строках вводится балл n того участника

Выходные данные

В единственной строке выведите сколько Максу нужно денег, чтобы поздравить всех призёров соревнования (количество призёров нужно умножить на 100 рублей)

Пример
Входные данные
5 1 2 3 4 5
Выходные данные
300
